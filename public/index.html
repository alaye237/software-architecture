Software Architecture

Node.js Express API with File-Based Persistence

Server is running on port 5000
All endpoints are available and functional.

Available Endpoints

POST /v1/api/users

Create a new user

GET /v1/api/users/:id

Get user by ID

POST /v1/api/auth/login

User login (authentication)

POST /v1/api/transactions

Create a transaction

GET /v1/api/transactions/user/:userId

Get user's transactions

Features

  • ✓ User management with secure password hashing
  • ✓ Transaction tracking and storage
  • ✓ File-based persistence (JSON storage)
  • ✓ CORS enabled for cross-origin requests
  • ✓ RESTful API architecture

Tech Stack

  • Node.js & Express.js
  • bcryptjs (password hashing)

  • CORS middleware
